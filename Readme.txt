# Node_With_Sequelize

1. Project's Title

Address book

2. How to Install and Run the Project

step 1:- write npm install on cmd.
step 2:- then you need to import node-with-sequelize.sql file on phpmyadmin.
step 3:- then you need to import Sequelize.postman_collection.json file on Postman.
step 4:- then you need to run project. write "node index.js" on cmd.
step 5:- then start postman and open imported Sequelize.postman_collection.json file and execute one by one apis.

3. Packages

1) bcrypt: The bcrypt is a password hashing technique used to build password security.
2) body-parser: allows express to read the body and then parse that into a Json object that we can understand.
3) dotenv: dotenv allows you to separate secrets from your source code.
4) express: Express provides methods to specify what function is called for a particular HTTP verb ( for example GET , POST ).
5) express-joi-validation: it is used to validate expressjs routes.
6) joi: Joi is a validation library that allows you to build schemas to validate JavaScript objects.
7) jsonwebtoken: It is used to genrate token with sign() method.
8) multer: It is used to uploading files.
9) mysql2: It is used to store data on database.
10) passport: Passport is Express-compatible authentication middleware for Node. js.
11) passport-jwt: Passport strategy for authenticating with a JSON Web Token.
12) sequelize: Sequelize is a powerful library in Javascript that makes it easy to manage a SQL database.
13) winston: It is used to logging errors on console as well as on .log file.

4. functionality

User have to register.
After register User can login.
When user logged-in User can view profile.
User can update profile.
User can create address book.
User can also create multiple address book at a time.
User can view address books.
User can update address book.
User can delete address book.
